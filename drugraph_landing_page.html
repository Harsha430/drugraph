Natural language input
"What are the risks of combining Warfarin, Aspirin, and Ibuprofen?"
Graph RAG results — 3 interaction pairs detected
Major

Warfarin + Aspirin

Concurrent use significantly increases bleeding risk. Aspirin inhibits platelet aggregation, compounding anticoagulant effect.

Major

Warfarin + Ibuprofen

NSAIDs displace warfarin from plasma proteins, elevating free warfarin concentration and hemorrhagic risk.

Moderate

Aspirin + Ibuprofen

Ibuprofen may interfere with Aspirin's cardioprotective effect when taken concurrently. Consider timing separation.

Features
Built for real clinical reasoning.
Core

Multi-hop graph traversal

Queries span Drug → Target → Disease → Contraindication in a single Cypher call. Finds risks that require 4+ hops — impossible with vector search.

Core

Schema-aware Cypher generation

LLM reads live Neo4j schema on every query. As data grows, query generation automatically adapts — zero manual prompt updates.

Core

Self-correcting query validation

Every generated Cypher runs through EXPLAIN before execution. On failure, the error is fed back to the LLM for automatic correction.

Intelligence

Safe alternative finder

Ask "alternatives to Ibuprofen that don't interact with Warfarin" — DruGraph traverses the category graph to find approved substitutes.

Intelligence

Patient drug profile risk map

Input a full medication list and get a severity-ranked interaction matrix — critical risks surface first, every pair checked.

Data

14,000+ drugs indexed

Sourced from DrugBank Open + RxNav API. Covers approved drugs, categories, molecular targets, side effects, and interaction severity levels.
